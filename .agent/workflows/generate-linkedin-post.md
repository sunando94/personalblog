---
description: Generate a high-engagement LinkedIn post for the current blog post
---

1. Read the content of the currently active blog post.
2. Refer to the **LinkedIn Post Generator** skill (`.agent/skills/linkedin-post-generator/SKILL.md`) for the strategy.
3. Generate a LinkedIn post draft that includes:
   - A compelling **Hook** (first 2 lines).
   - A **Core Value** summary of the blog post.
   - A recommendation for a **Visual** (e.g., a specific diagram or code snippet from the post).
   - A **Call to Action** (inviting comments).
   - The **"Link in Comments"** instruction.
   - 2-5 relevant **Hashtags**.
4. Present the draft to the user along with:
   - The recommended **Comment Text** (containing the link).
   - Advice on **Optimal Posting Time**.
   - A reminder about the **Golden Hour** engagement.
